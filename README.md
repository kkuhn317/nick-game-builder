# nick-game-builder

to run this, open it in command prompt and type `python -m http.server`

actually now run `python server.py`

go in a browser like Pale Moon with flash installed and go to `http://localhost:8000/play.html`

or: open flash player and run `http://localhost:8000/BuilderMain.swf?builderType=spongebob&dataPath=data/&slotID=-1` (prob wont work tho)

or this: `http://localhost:8000/BuilderMain.swf?v=2&builderType=spongebob&dataPath=data/&slotID=-1`




All the files we have available are archived here https://web.archive.org/web/*/https://www.nick.com/nick-assets/gamebuilder/*
and here https://archive.org/details/builder-main/. If there's another archive somewhere, please let me know!
